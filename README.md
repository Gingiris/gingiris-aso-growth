## 中文版

### 📖 简介

本指南整理自王恒加老师的 ASO 学习会议纪要（2026年3月），结合行业公开研究与最佳实践，提供了一个关于应用商店优化（ASO）与 App 冷启动的系统性、可执行的策略框架。

### 🎯 核心模块

| 模块 | 核心内容 |
|:---|:---|
| **ASO 基础** | 商店页面优化、关键词研究、评分优化、排行榜规则 |
| **冷启动策略** | Organic First 原则、渠道矩阵、App vs Web 差异 |
| **UGC 运营** | 创作者寻找与筛选、定价与激励、管理架构 |
| **多平台内容** | TikTok/Instagram/YouTube Shorts 差异与策略 |
| **AI 矩阵号** | AI 视频制作、账号养护、马甲包策略 |
| **平台对比** | App Store vs Google Play 算法规则差异 |

### 🔧 ASO 工具推荐

| 工具 | 定价 | 特点 |
|:---|:---|:---|
| [Astro](https://tryastro.app/) | ~$80-90/年 | 界面友好，适合独立开发者 |
| [点点数据](https://www.diandian.com/) | 按需 | 国内工具，全球数据 |
| [AppTweak](https://www.apptweak.com/) | 按需 | AI 智能洞察 |
| [AppFollow](https://appfollow.io/) | 按需 | 评论管理 + 关键词 |

### 🚀 冷启动核心原则：Organic First

```
Step 1: 创始人亲自下场 → 找到 Winning Format
Step 2: 复制成功模板 → 扩大 UGC 规模  
Step 3: 付费投流放大 → Meta 广告首选
```

### 📊 三大短视频平台对比

| 平台 | 用户年龄 | 付费转化率 | 核心指标 |
|:---|:---|:---|:---|
| **TikTok** | 13-18岁 | 最低 | 完播率、分享率 |
| **Instagram** | 18-25岁 | 中等 | Share Rate |
| **YouTube Shorts** | 25-35岁 | 最高 | 前3秒停留率 |

### 💰 UGC 定价参考

- **基础定价**：$10-20/条
- **激励机制**：固定费用 + 播放量阶梯奖励

## English Version

### 📖 Introduction

This playbook provides a comprehensive, actionable framework for App Store Optimization (ASO) and App cold start strategies. It covers everything from keyword research to UGC operations and multi-platform content strategies.

### 🎯 Core Modules

| Module | Key Content |
|:---|:---|
| **ASO Basics** | Store page optimization, keyword research, rating optimization |
| **Cold Start Strategy** | Organic First principle, channel matrix |
| **UGC Operations** | Creator sourcing, pricing & incentives |
| **Multi-Platform Content** | TikTok/Instagram/YouTube Shorts strategies |
| **AI Matrix Accounts** | AI video production, account nurturing |

### 🚀 Cold Start Core Principle: Organic First

```
Step 1: Founders Go First → Find Winning Format
Step 2: Replicate Success → Scale UGC
Step 3: Paid Amplification → Meta Ads First
```
